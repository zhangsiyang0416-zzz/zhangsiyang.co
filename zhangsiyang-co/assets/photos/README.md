# 照片放这里

先把你最想展示的照片复制到这个文件夹。

建议先准备这些文件名，网页会自动读取：

- `hero.jpg`：首页最大主图，最好是你本人或很有代表性的旅行照
- `city.jpg`：城市/建筑/街景
- `life.jpg`：生活感照片
- `travel-01.jpg`：旅行封面
- `life-01.jpg`：生活瞬间
- `city-01.jpg`：城市照片
- `food-01.jpg`：食物/餐桌
- `friends-01.jpg`：朋友/群像

照片建议：

- 横图、竖图都可以
- 单张尽量小于 2MB，网页会更快
- 不想公开的人脸先不要放
- 如果照片太多，先放 20 张，我再帮你挑和排版
